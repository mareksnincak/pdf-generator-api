Always use a regular dash (-) instead of an em dash (-) in all written content (docs, comments, ADRs, etc.).

**Why:** User preference - em dashes are not desired in this project's writing style.

**How to apply:** Any time writing prose in markdown files, code comments, or any other content, replace - with -. Also applies retroactively when editing existing files.
